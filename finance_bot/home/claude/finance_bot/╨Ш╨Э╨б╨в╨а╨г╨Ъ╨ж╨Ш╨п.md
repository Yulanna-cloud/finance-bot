# 🤖 Инструкция по запуску финансового бота

Читай по шагам. Каждый шаг — отдельная задача. Не спеши.

---

## ШАГ 1. Получить токен Telegram-бота (5 минут)

1. Открой Telegram, найди бота **@BotFather**
2. Напиши ему `/newbot`
3. Он спросит имя бота — напиши любое, например `Мои финансы`
4. Спросит username — напиши латиницей с `bot` на конце, например `yulannafinance_bot`
5. BotFather пришлёт токен вида: `7123456789:AAHxxx...xxxyyy`
6. **Скопируй и сохрани этот токен** — он нужен на шаге 5

---

## ШАГ 2. Получить Gemini API ключ (5 минут)

1. Открой в браузере: **https://aistudio.google.com/app/apikey**
2. Войди через свой Google аккаунт
3. Нажми **"Create API key"**
4. Выбери **"Create API key in new project"**
5. Скопируй ключ вида `AIzaSy...` — сохрани его

> ✅ Бесплатно, без карты. Лимит: 1500 запросов в день — тебе хватит с запасом.

---

## ШАГ 3. Настроить доступ к Google Sheets (10 минут)

Это немного сложнее, но один раз и навсегда.

### 3.1 Создать Service Account

1. Открой: **https://console.cloud.google.com/**
2. Нажми **"Select a project"** → **"New Project"**
3. Назови проект, например `finance-bot` → нажми **"Create"**
4. В меню слева найди **"APIs & Services"** → **"Library"**
5. Найди **"Google Sheets API"** → нажми **"Enable"**
6. Найди **"Google Drive API"** → нажми **"Enable"**
7. Иди в **"APIs & Services"** → **"Credentials"**
8. Нажми **"+ Create Credentials"** → **"Service account"**
9. Имя: `finance-bot` → нажми **"Create and continue"** → **"Done"**
10. Нажми на созданный Service Account в списке
11. Перейди на вкладку **"Keys"** → **"Add key"** → **"Create new key"**
12. Выбери **JSON** → нажми **"Create"**
13. Скачается файл `finance-bot-xxx.json` — **не удаляй и не публикуй его!**

### 3.2 Дать боту доступ к твоей таблице

1. Открой скачанный JSON-файл любым текстовым редактором (Блокнот)
2. Найди строку `"client_email"` — там будет адрес вида `finance-bot@xxx.iam.gserviceaccount.com`
3. **Скопируй этот email**
4. Открой свою Google Таблицу
5. Нажми **"Поделиться"** (кнопка в правом верхнем углу)
6. Вставь скопированный email
7. Выбери роль **"Редактор"**
8. Нажми **"Отправить"**

> ✅ Теперь бот может писать в твою таблицу!

## ШАГ 4. Загрузить код на GitHub (5 минут)

1. Зарегистрируйся на **https://github.com** (если нет аккаунта)
2. Нажми **"+"** → **"New repository"**
3. Назови `finance-bot`, поставь **Private** (приватный!)
4. Нажми **"Create repository"**
5. На следующей странице нажми **"uploading an existing file"**
6. Перетащи туда ВСЕ файлы из папки `finance_bot`:
   - `bot.py`
   - `requirements.txt`
   - `Procfile`
   - `railway.toml`
   - Папку `handlers` (со всеми файлами внутри)
   - Папку `services` (со всеми файлами внутри)
7. Нажми **"Commit changes"**

---

## ШАГ 5. Запустить на Railway (10 минут)

1. Открой **https://railway.app**
2. Нажми **"Login"** → войди через GitHub
3. Нажми **"New Project"** → **"Deploy from GitHub repo"**
4. Выбери репозиторий `finance-bot`
5. Railway начнёт деплой — подожди минуту

### Добавить переменные окружения (очень важно!)

6. Нажми на созданный сервис → вкладка **"Variables"**
7. Нажми **"+ New Variable"** и добавь три переменные:

**Переменная 1:**
```
Name:  TELEGRAM_BOT_TOKEN
Value: (вставь токен из Шага 1)
```

**Переменная 2:**
```
Name:  GEMINI_API_KEY
Value: (вставь ключ из Шага 2)
```

**Переменная 3:**
```
Name:  GOOGLE_CREDENTIALS_JSON
Value: (открой JSON-файл из Шага 3 в Блокноте, выдели ВЕСЬ текст и вставь)
```

8. После добавления переменных Railway автоматически перезапустит бота
9. Во вкладке **"Logs"** должно появиться: `Бот запущен!`

---

## ШАГ 6. Проверить что всё работает

1. Найди своего бота в Telegram по username из Шага 1
2. Напиши `/start` — он должен ответить приветствием
3. Напиши голосовое: _«потратила на кофе триста рублей»_
4. Проверь свою таблицу — в листе ОПЕРАЦИИ должна появиться новая строка!

## Как пользоваться

### Голосовые сообщения 🎤
Просто говори как обычно:
- «купила воду за двадцать рублей»
- «потратила в Пятерочке тысячу четыреста»
- «заплатила за такси триста»

### Фото чека 📷
- Сфотографируй чек (нужен хороший свет, чек должен быть чётким)
- Отправь фото в бот
- Бот прочитает ВСЕ позиции и запишет каждую отдельно

### Банковская выписка 📄
**Сбербанк:**
1. Приложение Сбербанк → История операций
2. Фильтр по периоду → Экспорт → CSV

**Т-Банк:**
1. Приложение → История → Выгрузить выписку → Excel

Отправь файл боту — он прочитает и запишет все операции.

### Отчёт 📊
Напиши `/отчет` — получишь:
- Сумму доходов и расходов за месяц
- Топ-5 категорий расходов
- Прогноз на месяц

---

## Если что-то не работает

**Бот не отвечает** → проверь Logs на Railway, там будет написано что случилось

**Ошибка Google Sheets** → убедись что дал доступ service account (Шаг 3.2)

**Ошибка Gemini** → проверь правильность GEMINI_API_KEY в Variables

**Голос не расшифровывается** → Gemini иногда не поддерживает ogg, попробуй текстом
